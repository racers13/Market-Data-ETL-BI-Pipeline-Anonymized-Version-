# 📊 Market Data ETL & BI Pipeline (Anonymized Version)

ETL-пайплайн для сбора рыночных данных, имитации бизнес-показателей и аналитики.

## 🚀 О проекте
Система автоматизирует полный цикл работы с данными: от многопоточного парсинга различных торговых площадок до визуализации ключевых метрик. В проекте реализована сложная бизнес-логика имитации рыночной активности, что позволяет использовать его для тестирования BI-инструментов.

### Ключевые модули:
- **Data Extraction:** Парсеры на базе Scrapy и BeautifulSoup4 для сбора данных из категорий "Металлопрокат" и "Книги" (источники обезличены).
- **Data Transformation:** Унификация разнородных атрибутов товаров в единый стандарт.
- **Business Logic Engine:** 
    - Генерация динамических продаж (каждые 30 минут).
    - Расчет выручки и маржинальности.
    - Моделирование поступлений товара и управление остатками.
- **Price Recommendation System:** Алгоритм, предлагающий изменение цены на основе дефицита или низкого спроса.
- **Automation:** Планировщик задач (APScheduler) для автономной работы 24/7.

## 🛠 Стек технологий
- **Python 3.10+**
- **PostgreSQL 18** (SQLAlchemy / Psycopg2)
- **Scrapy / BeautifulSoup4** (Web Scraping)
- **Pandas** (Data Processing)
- **APScheduler** (Task Scheduling)
- **Yandex DataLens** (BI Visualization)

## 🏗 Структура проекта
- `parser_*.py` — модули сбора данных (обезличенные URL).
- `models.py` — универсальная модель данных товара.
- `database.py` — слой взаимодействия с PostgreSQL.
- `business_logic.py` — ядро бизнес-расчетов и имитации продаж.
- `scheduler.py` — диспетчер автоматического запуска задач.

## ⚙️ Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Настройте параметры подключения в `database.py` (используйте заглушку `YOUR_POSTGRES_PASSWORD`).
4. Запустите систему: `python scheduler.py`
